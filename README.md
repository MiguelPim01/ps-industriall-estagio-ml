# Processo seletivo IndustriALL

Projeto desenvolvido como parte do processo seletivo de estágio para a IndustriALL.

## Instalação

Utilize o gerenciador de pacotes `uv` para instalar e gerir as dependências deste projeto.

Abra o terminal na raiz do projeto e rode o comand:
```bash
uv sync
```

## Rodando o projeto

### Adição dos dados

Adicione todos os arquivos `.csv` dentro da pasta `data/` e o programa já irá ler e processar todos os dados.
